 - HTML5 画布 Canvas

## 游戏简介
 
 移动端2D飞行射击游戏

 触摸飞机移动躲避敌机
 
 道具：

 - 炸弹（清屏，点触左下使用，上限3枚）
 - 武器（20s强化，可叠加4次）
 
 分数：

 - 小型敌机精灵：100分
 - 中型敌机精灵：1200分
 - 大型敌机精灵：3500分
 - 智能敌机精灵：双倍得分
 
 左上点触暂停

 电脑版chrome 需要F12切换为移动端页面

 移动端chrome 无法取消下滑默认浏览器移动事件 在chrome://flags中可设置
